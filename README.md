## Controls

### Moving the object
- `W` - up
- `S` - down
- `A` - left
- `D` - right

### changing zoom and depth
- &uarr; - zoom in
- &darr; - zoom out
- &larr; - depth up
- &rarr; - depth down

### changing projection
- `P` + `1` - normal projection
- `P` + `2` - fisheye projection
- `P` + `3` - curvliniear projeciton
- `P` + `I` - partymode (seizure warning!)
- `P` + `O` - partymode off

### how to use
- clone the repo
- type `./fdf /test_maps/any_map.fdf`
